# AI辅助固收研究系列报告 第6篇：T06-城投区域分析

> **研究领域**: 城投债/信用债
> **研究目的**: 建立系统化的城投区域风险评级体系，结合利差分位数提供择时信号，实现从风险识别到投资策略的闭环支持
> **适用场景**: 投资组合区域配置、新发债券定价参考、存量持仓风险监控
> **更新频率**: 按需/季度更新

---

## 1. 研究背景与问题

### 1.1 为什么需要这项研究？

城投债是中国债券市场最具特色的品种之一，其信用风险与地方政府财政状况深度绑定。2023年9月"一揽子化债"政策启动以来，城投债市场经历了深刻变革：

**政策环境剧变**
- 6万亿元地方债务限额用于置换存量隐性债务，为城投平台提供了系统性化债支持
- 2025年已有362家城投平台完成"退平台"，城投主体正在加速市场化转型
- 城投债净融资降至百亿级，供给端持续收缩
- 2027年6月被设定为化债最终时限，时间窗口明确

**投资逻辑重构**
"刚兑信仰"时代已经结束，城投债投资从"信仰驱动"转向"区域定价"。同一评级、同一期限的城投债，因所在区域不同，利差可能相差数百个基点。区域财政实力和债务负担成为最核心的风险定价因子。

在这一背景下，传统的"看名字买债"、"看评级买债"的投资方式已经失效。投研人员需要建立一套**系统化、可量化、可跟踪**的区域风险评级体系，才能在复杂的城投债市场中做出理性的投资决策。

### 1.2 核心研究问题

本项目旨在回答以下研究问题：

- **问题一：如何系统化评估不同区域的信用风险？** 传统的人工分析难以覆盖全国数百个发债区域，需要建立一套可复制的评级框架。
- **问题二：区域风险如何传导至平台层面？** 城投平台与地方政府存在复杂的关联关系，如何将区域风险内化为平台评级的约束条件？
- **问题三：评级结果如何转化为投资策略？** 信用评级只是投资决策的起点，如何结合市场信号（如利差分位数）生成可操作的投资建议？

### 1.3 研究价值

本项目的核心价值在于**提供从风险识别到投资策略的闭环支持**：

- **风险识别**：通过五大维度的量化评分，系统化识别各区域的信用风险暴露
- **风险传导**：将区域评级作为平台评级的核心输入，实现宏观风险向微观主体的传导
- **策略生成**：结合利差分位数择时信号，将评级结果转化为具体的仓位、久期和风控建议

## 2. 研究思路与方法

### 2.1 整体研究框架

本项目的分析框架可概括为"**一体系、双层级、三维度**"：

```
┌─────────────────────────────────────────────────────────────┐
│                    投资决策闭环                              │
├─────────────────────────────────────────────────────────────┤
│  ┌───────────────┐    ┌───────────────┐    ┌─────────────┐ │
│  │  区域评级      │ ─→ │  平台评级      │ ─→ │ 投资策略    │ │
│  │  (风险识别)    │    │  (风险传导)    │    │ (决策输出)  │ │
│  └───────────────┘    └───────────────┘    └─────────────┘ │
│         ↓                     ↓                    ↓        │
│   区域风险分           平台综合评级          仓位/久期/风控  │
│   布与建议                                  择时信号         │
└─────────────────────────────────────────────────────────────┘
```

**一体系**：统一的评级方法论，确保区域评级与平台评级的可比性和一致性。

**双层级**：区域层级和平台层级的双层评级，覆盖宏观和微观两个维度。

**三维度**：风险识别（买什么）、风险传导（传导机制）、策略生成（何时买、买多少）。

### 2.2 核心方法论

#### 2.2.1 区域评级：五维度加权评分体系

区域评级采用五维度加权评分方法，总分100分，各维度权重如下：

| 维度 | 权重 | 核心逻辑 |
|------|------|----------|
| 财政实力 | 30% | 决定偿债能力的核心因子，包括一般预算收入、财政自给率、上级补助等 |
| 债务负担 | 30% | 决定偿债压力的核心因子，包括债务率、负债率、债务结构等 |
| 经济活力 | 20% | 财政可持续性的支撑因子，包括GDP增速、产业结构、人口流动等 |
| 金融支持 | 10% | 融资便利性的辅助因子，包括金融机构存款、贷款余额等 |
| 政策环境 | 10% | 特殊支持的辅助因子，包括化债试点、专项政策等 |

**权重设计的核心理念**：财政实力和债务负担合计占比60%，体现了"财政债务为核心"的风险定价理念。这一设计与中证鹏元、联合资信等主流评级机构的框架保持一致——财政实力决定偿债能力，债务负担决定偿债压力。

#### 2.2.2 平台评级：区域风险锚定的五维度体系

平台评级同样采用五维度评分，但引入"区域风险"作为核心输入：

| 维度 | 权重 | 核心逻辑 |
|------|------|----------|
| 股东背景 | 25% | 与政府的关联程度，包括股权层级、实际控制人等 |
| 区域风险 | 25% | 直接引用区域评级结果，实现风险传导 |
| 财务状况 | 25% | 平台自身的偿债能力，包括资产负债率、现金流等 |
| 市场表现 | 15% | 市场对平台的定价，包括利差水平、成交活跃度等 |
| 增信情况 | 10% | 外部支持的可得性，包括担保、抵押等 |

**关键设计**：区域风险权重设为25%，与股东背景、财务状况并列。这意味着即使平台自身财务指标优秀，若所在区域风险较高，其综合评级也会受到限制。这种设计符合城投平台"地方政府的延伸"这一本质特征。

#### 2.2.3 利差分位数择时信号

评级解决"买什么"的问题，但投资决策还需要回答"何时买"。本项目引入利差分位数作为择时信号：

| 利差分位数 | 信号含义 | 投资建议 |
|------------|----------|----------|
| >80% | 利差处于历史高位，市场定价悲观 | 超配信号，可积极布局 |
| 50%-80% | 利差处于中性偏高区间 | 标配，关注边际变化 |
| 20%-50% | 利差处于中性偏低区间 | 谨慎配置，控制久期 |
| <20% | 利差处于历史低位，市场定价乐观 | 减仓信号，逐步退出 |

这一择时策略与浙商证券研报的验证结果相呼应：7年和10年AA+城投债利差触及上轨时择机增配，历史择时超额收益分别为2.06%和3.99%。

### 2.3 与传统研究方式的对比

**传统方式**：
- 依赖分析师的主观判断和经验积累
- 各区域分析框架不统一，难以横向比较
- 评级结果难以量化回溯和验证
- 策略建议缺乏明确的触发条件

**技术辅助后**：
- 建立标准化的五维度评级框架，确保分析的一致性
- 区域评级与平台评级联动，实现风险传导的量化
- 利差分位数提供客观的择时信号，减少主观判断
- 配置化的权重和阈值设计，支持根据市场环境快速迭代

**主要改进**：
- 从"个案分析"升级为"系统化评级"，覆盖范围和效率显著提升
- 从"定性判断"升级为"定量评分"，评级结果可比较、可回溯
- 从"单一维度"升级为"评级+择时双因子"，投资决策更完整

## 3. 数据来源与处理

### 3.1 数据来源说明

| 数据类型 | 来源 | 更新频率 | 用途 |
|---------|------|---------|------|
| 区域财政数据 | Wind/中债登/地方财政年鉴 | 年度/季度 | 财政实力、债务负担评分 |
| 区域经济数据 | Wind/统计局 | 季度/年度 | 经济活力评分 |
| 区域金融数据 | Wind/人行统计 | 季度 | 金融支持评分 |
| 城投平台财务数据 | Wind/评级报告 | 年度/季度 | 平台财务状况评分 |
| 城投债成交利差 | Wind/iFinD | 日度 | 利差分位数计算 |
| 政策文件 | 政府官网/券商研报 | 不定期 | 政策环境评分 |

### 3.2 数据处理要点

**财政实力指标计算**
- 一般预算收入：采用三年平均值，平滑年度波动
- 财政自给率 = 一般预算收入 / 一般预算支出
- 上级补助依赖度 = 上级补助收入 / 综合财力

**债务负担指标计算**
- 债务率 = 政府债务余额 / 综合财力
- 负债率 = 政府债务余额 / GDP
- 债务结构：关注短期债务占比和隐性债务规模

**数据质量校验**
- 对异常值进行标记和人工复核
- 缺失数据采用插值或同类区域均值填充
- 定期与Wind官方数据进行交叉验证

## 4. 核心分析框架

### 4.1 分析维度一：区域风险评级

**分析内容**：从财政实力、债务负担、经济活力、金融支持、政策环境五个维度，对全国发债区域进行综合评分。

**关键发现**：
- 财政实力和债务负担是区域风险的核心决定因素，两者的得分与区域城投债利差呈现高度相关性
- 经济活力对区域风险的边际影响显著，经济增长放缓的区域往往面临财政实力和债务负担的双重压力
- 政策环境（如化债试点）对特定区域的评级有显著的提升作用，但影响程度取决于政策落地的实质性

**业务含义**：
- 高评级区域（AAA/AA+）可作为投资组合的核心配置，提供稳定的收益和较低的信用风险
- 中评级区域（AA）需要结合利差分位数判断是否具有配置价值，控制单个区域的集中度
- 低评级区域（AA-及以下）建议谨慎参与或回避，除非有充分的利差补偿

### 4.2 分析维度二：平台风险评级

**分析内容**：在区域评级的基础上，结合平台自身的股东背景、财务状况、市场表现、增信情况，对城投平台进行综合评分。

**关键发现**：
- 区域风险评分对平台综合评级具有显著的约束作用：即使财务状况优秀的平台，若区域评级较低，综合评级也会受到限制
- 股东背景（股权层级、实际控制人）对平台获得政府支持的确定性有直接影响
- 市场表现（利差水平、成交活跃度）可以作为评级结果的验证：利差异常偏低可能提示评级偏高，利差异常偏高可能提示评级偏低

**业务含义**：
- 平台评级应优先选择区域评级高且自身财务稳健的"双优"主体
- 对于区域评级一般但平台自身优秀的"单优"主体，应控制配置比例和久期
- 对于区域评级和平台评级均较低的主体，建议回避或仅作为高收益策略的标的

### 4.3 分析维度三：投资策略生成

**分析内容**：将区域评级、平台评级与利差分位数结合，生成具体的投资策略建议。

**策略生成规则**：

| 评级 | 利差分位数 | 策略类型 | 仓位建议 | 久期建议 | 择时信号 |
|------|------------|----------|----------|----------|----------|
| AAA | >80% | 积极配置 | 可超配至15% | 3-5年 | 强买入 |
| AAA | 50-80% | 标准配置 | 维持10% | 2-3年 | 持有 |
| AA+ | >80% | 机会配置 | 可增至10% | 2-3年 | 买入 |
| AA+ | 20-50% | 谨慎配置 | 维持5% | 1-2年 | 观望 |
| AA | >80% | 博弈配置 | 可增至5% | 1-2年 | 谨慎买入 |
| AA | <20% | 减仓 | 降至2% | <1年 | 卖出 |
| AA-及以下 | 任意 | 回避 | 0% | - | 不参与 |

**业务含义**：
- 评级决定配置基础：高评级区域/平台提供稳定的配置盘，低评级区域/平台需要更谨慎的风险管理
- 利差分位数决定择时方向：高利差分位数提供买入机会，低利差分位数提示卖出风险
- 仓位和久期的动态调整：根据评级和择时信号的综合判断，灵活调整投资组合的结构

## 5. 技术/AI辅助要点

本项目的技术设计主要体现了以下特色：

**配置化设计**
评级权重、利差阈值、策略规则等核心参数均在配置文件中定义，支持根据市场环境变化快速调整。例如，在经济下行期可以适当提高财政实力的权重，在政策宽松期可以降低政策环境的敏感度。

**双层评级联动**
区域评级结果直接作为平台评级的输入（权重25%），实现了从宏观到微观的风险传导。这种设计避免了区域分析和平台分析的割裂，确保了评级体系的内在一致性。

**利差分位数的动态计算**
利差分位数基于滚动窗口计算，窗口期可配置（默认3年）。这种方式既保证了择时信号的稳定性，又能及时反映市场环境的变化。

## 6. 心得要点

### 要点一：评级与择时的双因子决策框架

**背景**：传统的城投债分析往往只关注信用评级或只关注市场利差，两者割裂导致投资决策不完整。只看评级可能错过市场错误定价的机会，只看利差可能忽视基本面恶化的风险。

**做法**：采用"评级+利差分位数"的双因子框架：评级解决"买什么"（信用风险定价），利差分位数解决"何时买"（市场时机选择）。具体而言，高评级（AAA/AA+）区域提供核心配置基础，而利差分位数>80%时提供超配信号，<20%时提示减仓风险。

**洞察**：投资决策需要同时回答"买什么"和"何时买"两个问题。单一维度的分析框架存在盲点：只看评级可能错过市场错误定价的机会，只看利差可能忽视基本面恶化的风险。双因子框架通过交叉验证，提高了投资决策的完整性和可靠性。

这种方法论可以扩展到其他信用品种的分析，如产业债、地产债等。核心逻辑是：**评级框架解决基本面问题，市场信号解决择时问题，两者缺一不可**。

**关联**：与"模块化设计"、"数据驱动验证"等原则相通——通过将评级和择时解耦为独立模块，再通过组合规则生成决策，提高了框架的灵活性和可维护性。

### 要点二：区域风险作为平台评级的锚定因子

**背景**：城投平台的信用风险本质上源于其与地方政府的关联关系。如何量化这种关联关系，使平台评级既能反映自身特点，又能体现区域风险传导？

**做法**：在平台评级体系中，将"区域风险"作为五大维度之一，权重设为25%，与股东背景、财务状况并列。区域风险评分直接来源于区域评级结果，实现了两个评级体系的有机联动。

**洞察**：系统化分析的关键在于识别和量化核心传导路径。城投平台的风险传导路径是：区域财政状况 -> 区域偿债能力 -> 平台获得政府支持的可能性 -> 平台信用风险。将区域风险作为平台评级的锚定因子，实质上是将宏观风险内化为微观评级的约束条件。

这种"宏观约束微观"的设计思路，可以推广到任何存在系统性风险传导的场景。例如，在地产债分析中，可以将城市风险（房价走势、土地市场）作为企业评级的约束因子；在银行债分析中，可以将区域经济风险作为银行评级的约束因子。

**关联**：与"模块化设计"、"工程化思维"等原则相通——通过识别和量化风险传导路径，将复杂的系统性风险分解为可处理的模块。

### 要点三：权重设计的"2-2-1"黄金比例

**背景**：区域评级涉及多个维度，如何确定各维度的权重？权重设置过主观会影响评级的公信力，完全依赖数据又可能忽视行业经验。

**做法**：采用"2-2-1"的权重分配：财政实力和债务负担各占30%（合计60%），经济活力占20%，金融支持和政策环境各占10%。这种设计借鉴了中证鹏元、联合资信等主流评级机构的框架，同时结合了城投债的特殊性——财政债务是核心，经济增长是支撑，金融和政策是辅助。

**洞察**：多维度评价体系的权重设计应遵循"核心-支撑-辅助"的层次结构。核心维度（财政债务）应占据主导权重（>50%），支撑维度（经济活力）提供边际贡献（15-25%），辅助维度（金融政策）用于捕捉特殊情况（<20%）。

这种层次化的权重设计，既体现了风险定价的主次逻辑，又保留了应对特殊情况的灵活性。权重设计不是追求"精确"，而是追求"合理且可解释"——能够清晰说明为什么某个维度更重要，以及在什么情况下权重可能需要调整。

**关联**：与"数据驱动验证"、"迭代优化"等原则相通——初始权重基于行业经验和主流框架设定，后续可通过历史数据回测和市场验证进行迭代优化。

## 8. 延伸思考

### 8.1 可扩展的研究方向

**动态权重调整**
当前权重为固定值，未来可根据市场环境实现动态调整。例如，在经济下行期提高财政实力的权重，在政策宽松期提高政策环境的敏感度。

**评级预警机制**
当前框架主要提供时点评级，缺乏评级变化的预警机制。未来可引入财政数据的边际变化、利差的异常波动等信号，建立评级预警体系。

**跨区域风险传染分析**
城投区域之间存在一定的风险传染效应（如担保链、跨区域投资等），未来可建立区域间的风险传染模型，识别系统性风险的传导路径。

### 8.2 与其他项目的关联

**与T07（信用利差分析）的关联**：本项目使用的利差数据可与T07的利差分析框架对接，实现更精细的利差归因分析。

**与T08（城投债择时策略）的关联**：本项目的评级结果可作为T08择时策略的输入，结合技术指标和基本面信号生成综合择时建议。

**与T09（信用风险预警）的关联**：本项目的评级框架可与T09的预警模型结合，实现从评级到预警的闭环。

### 8.3 待改进之处

**数据依赖性**：评级质量高度依赖输入数据的准确性和完整性，若区域财政数据存在滞后或失真，评级结果会产生偏差。

**权重静态性**：当前权重设置为固定值，未实现根据市场环境动态调整。

**政策因素量化难度**：政策环境评分目前依赖主观判断，缺乏客观量化指标，可能引入人为偏差。

**利差数据可得性**：部分弱资质区域城投债流动性不足，利差数据可能失真或不连续，影响择时信号的可靠性。
