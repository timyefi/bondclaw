# T45-曲线 - 债券收益率曲线计算系统

## 项目概述

本项目是一个用于构建债券收益率曲线的数据处理和分析工具。该系统能够基于市场债券数据，通过 **Hermite 插值法** 计算任意期限的收益率曲线，支持城投债、产业债、二永债（二级资本债、永续债）等多种债券类型，并按照发行方式、是否永续、是否次级、信用评级等多个维度进行分类计算。

## 核心功能

1. **收益率曲线构建**：基于中债曲线数据，使用 Hermite 插值法计算任意期限的收益率
2. **多维度分类**：支持城投/产业、公募/私募、永续/非永续等多种分类维度
3. **信用债利差计算**：计算信用债相对于基准曲线的利差
4. **标准化收益率调整**：将任意期限债券的收益率调整到标准期限

## 核心算法

### Hermite 插值法

使用三次 Hermite 插值计算任意期限的收益率：

```
H1(t) = 1 - 3t^2 + 2t^3
H2(t) = 3t^2 - 2t^3
H3(t) = t - 2t^2 + t^3
H4(t) = -t^2 + t^3

y(x) = y1*H1(t) + y2*H2(t) + d1*H3(t) + d2*H4(t)
```

其中 `t = (x - x1) / (x2 - x1)`

## 关键设计决策

1. **Hermite 插值选择**：保持曲线光滑（导数连续），避免龙格现象
2. **4个锚点插值**：使用中心差分计算导数，提高精度
3. **债券余额加权**：反映市场真实情况
4. **期限差异容忍度**：2年，平衡数据覆盖率和准确性

- 边界情况（如30年附近）需要特殊处理
