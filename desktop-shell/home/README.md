# Home Panel

This directory holds the first-screen model for the BondClaw desktop shell.

## Files

- `home_model.py`

## Purpose

The home panel should combine the local desktop settings, runtime summary,
provider status, prompt shortcuts, research subscriptions, contact collection
state, and the support banner into one renderable model.
