# BondClaw V1 Windows 原生安装与运行说明

这是给最终客户看的最短说明。

## 安装前

- 只需要一台可正常使用的 Windows 电脑。
- 不需要先装 Linux 发行版。
- 不需要单独配置终端环境。

## 安装后

1. 双击安装包完成安装。
2. 启动 BondClaw。
3. 进入 BondClaw 首页。
4. 需要时再打开模板中心、信息中心、联系方式和设置。

## 使用时

- 首页负责总览。
- 模板中心负责按角色浏览模板。
- 信息中心负责浏览订阅和案例。
- 联系方式负责查看本地队列和投递状态。
- 设置负责调整默认角色、默认模板、provider 和本地行为。

## 客户侧原则

- 客户不需要切换到 bash。
- 客户只需要按 Windows 桌面应用的方式使用 BondClaw。

## 常见问题

- 如果无法启动，先检查安装是否完整。
- 如果页面空白，先重启应用再看日志。
- 如果升级后状态异常，回退到上一个可用版本。

## 交付目标

- 安装简单。
- 启动简单。
- 页面结构固定。
- 执行层对客户透明。
