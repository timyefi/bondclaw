# T46 - 曲线优化

## 项目概述

本项目实现曲线计算系统的优化版本，主要用于债券标准化收益率的计算。

## 功能特性

### 1. 更精细的目标期限粒度
- 从18个标准点提升到0.01年精度（约3000个点）
- 支持更精确的期限匹配和插值

### 2. 针对性目标期限
聚焦7个关键期限点：
- 0.5年（6个月）
- 1年
- 1.75年（21个月）
- 2年
- 3年
- 4年
- 5年

### 3. 数据结构优化
- 按目标期限拆分DataFrame
- 便于分别存储和处理

## 核心算法

### Hermite插值
使用Hermite插值（中债插值公式）计算非标准期限点的收益率。

### 标准化收益率计算
```
标准化收益率 = 中债行权估值 + (目标期限收益率 - 原期限收益率)
```

## 使用方法

1. 打开 `重构.ipynb`
2. 按顺序执行各个单元格
3. 主要流程：
   - 环境配置
   - 数据获取
   - 数据处理
   - 核心计算（匹配、插值、标准化）
   - 结果输出

## 数据流程

1. **数据加载**：从数据库加载市场数据和曲线数据
2. **数据修正**：修正隐含评级与城投的对应关系
3. **债券分类**：按发行方式、是否永续等分类
4. **收益率匹配**：
   - 目标期限曲线收益率匹配
   - 原期限曲线收益率匹配
5. **Hermite插值**：对非标准期限点进行插值
6. **标准化计算**：计算标准化收益率
7. **结果拆分**：按目标期限拆分DataFrame
