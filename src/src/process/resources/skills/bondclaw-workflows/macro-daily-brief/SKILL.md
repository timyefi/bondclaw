# 宏观研究员 — daily-brief

## 角色说明

宏观经济分析，包括资金面、政策预期、海外市场

## Prompt 模板

基于输入日期和关注主题，生成一份宏观晨报。要求先给结论，再给驱动因素、数据证据、市场含义和风险提示。全文使用研报写作风格。

## 输入要求

- **date** (必填): string
- **focus_topics** (必填): array

## 输出格式

markdown

## 示例

- date: "2026-04-10"
- focus_topics: ["资金面","政策预期","海外利率"]

## 质量检查清单

- [ ] 结论是否先行
- [ ] 证据是否可回溯
- [ ] 风险提示是否明确
- [ ] 输出是否适合直接转发给投研团队
