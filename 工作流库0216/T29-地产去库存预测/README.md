# T29: 地产去库存预测

## 项目概述

本项目基于EDB数据库中的房地产相关时间序列数据，对住宅库存进行预测分析，提供两种预测方法：

1. **趋势推演预测**：基于历史趋势的线性回归
2. **销售-竣工去化预测**：基于供需平衡的动态预测

## 业务背景

- 跟踪中国房地产市场住宅库存变化
- 预测未来库存水平（预测到2027年12月）
- 计算库销比指标，评估市场去化压力

## 核心算法

### 1. 住宅库存估算

```python
# 住宅库存估算公式
# 住宅库存估算值 = 待售面积 * (期房累计 + 现房累计) / 期房累计
inventory_estimate = df['S0029674'] * \
    (df_monthly['S0049296_hist_cum'] + df_monthly['S0049264_hist_cum']) / \
    df_monthly['S0049296_hist_cum']
```

### 2. 趋势线性回归预测

基于历史数据的线性回归趋势进行未来值预测。

### 3. 销售-竣工去化预测

基于供需平衡的动态预测：库存变化 = 销售 - 竣工

### 4. 库销比计算

```python
# 库销比 = 库存 / (年度销售/12)
库销比 = 库存预测口径 / (住宅销售12个月滚动 / 12)
```

## 注意事项

1. **数据处理**：
   - 年度累计值需转换为月度值
   - 年初数据需要特殊处理（1月份归零）
   - 数据为月度频率，每月最后一天

2. **预测逻辑**：
   - 趋势预测基于线性回归假设
   - 销售-竣工预测基于供需平衡
   - 预测结果仅供参考

3. **配置调整**：
   - 预测开始日期默认为当前日期3年前
   - 可在config.py中修改预测参数
