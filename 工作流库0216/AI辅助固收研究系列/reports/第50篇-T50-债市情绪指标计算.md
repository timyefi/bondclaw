# AI辅助固收研究系列报告 第50篇：T50-债市情绪指标计算

> **研究领域**: 利率债/市场情绪
> **研究目的**: 构建中国债券市场的情绪指标
> **适用场景**: 日常跟踪/市场情绪监测
> **更新频率**: 每日

---

## 1. 研究背景与问题

### 1.1 为什么需要这项研究？

在金融市场中，情绪指标是投资者判断市场状态、预测行情转折的重要工具。股票市场拥有成熟完善的情绪指标体系，其中最著名的当属VIX恐慌指数，它通过标普500指数期权的隐含波动率来衡量市场对未来30天波动性的预期，被广泛视为市场恐慌程度的"温度计"。

然而，债券市场的情绪指标体系相对匮乏。虽然国际市场存在MOVE指数——被称为"债券市场的VIX"，它通过追踪2年、5年、10年、30年等多个期限利率互换期权的隐含波动率来衡量美债市场波动性，但这类指标高度依赖发达的衍生品市场。对于中国债券市场而言，期权市场尚不成熟，相关数据的可得性受到较大限制。

国内市场上，机构研究者们也开发了多种债市情绪指标，主要包括：
- **国开债隐含流动性溢价**：通过国开债与国债的利差来间接反映市场流动性偏好
- **利率债投标情绪**：观察一级市场投标倍数和边际利率来衡量配置需求
- **隔夜回购比例**：反映市场杠杆水平和资金面紧张程度
- **持仓多空集中度**：通过期货市场的持仓结构来判断多空力量对比

这些指标各有侧重，但往往需要复杂的数据源支持，或仅能反映市场的某一侧面。对于日常投研工作而言，一个简单、直观、易于计算的情绪指标仍然具有很高的实用价值。

### 1.2 核心研究问题

本项目旨在回答以下研究问题：

1. **能否仅用现货市场数据构建有效的债市情绪指标？** 在缺乏发达衍生品市场的环境下，是否可以通过创造性地组合现货数据来捕捉市场情绪？

2. **如何区分债市中"乐观放量"与"恐慌抛售"两种截然不同的状态？** 同样是成交量放大，收益率上升和下降所代表的市场情绪完全相反，如何在指标设计中体现这种差异？

3. **如何平衡指标的灵敏度与稳定性？** 情绪指标需要及时反映市场变化，但又不能过度敏感导致噪音干扰。

### 1.3 研究价值

本研究的价值体现在以下几个方面：

**降低数据门槛**：仅需要10年国债收益率和成交金额两个数据源，无需期权数据或复杂的衍生品定价模型，适用于各类市场参与者。

**创新性方法论**：通过方向系数的设计，将量价关系转化为情绪信号，能够有效区分市场的乐观与恐慌状态。

**实用性强**：提供全量计算和增量更新两种运行模式，既可用于历史回溯分析，也可嵌入日常监控流程。

**可扩展性好**：核心参数可配置，框架可扩展至更多数据源，为后续优化预留空间。

## 2. 研究思路与方法

### 2.1 整体研究框架

本研究采用"双维度方向性情绪指标"框架，核心思想是将成交金额变化与收益率变动方向相结合，通过方向系数的调整来捕捉债市情绪的本质特征。

整体研究框架如下：

```
数据输入层                    计算处理层                    输出层
┌──────────────┐           ┌──────────────┐           ┌──────────────┐
│ 10年国债收益率│──────────▶│ 收益率增长率  │──────────▶│              │
└──────────────┘           └──────────────┘           │              │
                                                      │  情绪指标    │
┌──────────────┐           ┌──────────────┐           │  (sentiment) │
│  成交金额     │──────────▶│ 成交金额增长率│──────────▶│              │
└──────────────┘           └──────────────┘           └──────────────┘
                                  │
                           ┌──────▼──────┐
                           │  方向系数    │
                           │ (direction) │
                           └─────────────┘
```

### 2.2 核心方法论

#### 2.2.1 情绪指标公式

情绪指标的计算公式为：

```
sentiment_index = yield_weight × yield_growth_rate + amount_growth_rate × direction_factor
```

其中：
- `yield_weight`：收益率权重，默认为0.5
- `yield_growth_rate`：近20个交易日10年国债收益率的变化率
- `amount_growth_rate`：近20个交易日成交金额的变化率
- `direction_factor`：方向系数，根据收益率变动方向确定

#### 2.2.2 核心组件详解

**（1）成交金额增长率**

计算公式：`(当前成交金额 - 起始成交金额) / 起始成交金额`

这一指标反映市场活跃度的变化。成交金额增加意味着市场参与度提高，可能预示着重要行情的酝酿或确认。

**（2）收益率增长率**

计算公式：`(当前收益率 - 起始收益率) / 起始收益率`

这一指标反映利率变动的趋势和幅度。在债市中，收益率上升意味着价格下跌，反之亦然。

**（3）方向系数——核心创新**

方向系数是本项目的核心创新点，其设计逻辑如下：

| 收益率变动方向 | 方向系数 | 经济含义 |
|---------------|---------|---------|
| 收益率上升 | +1 | 乐观放量：投资者在收益率上升时继续买入，追逐收益 |
| 收益率下降 | -1 | 恐慌抛售：投资者在收益率下降时恐慌性卖出，规避风险 |
| 收益率不变 | 0 | 市场观望：成交量变化无明确方向信号 |

这一设计的核心洞察在于：**债市中成交量增加的情绪含义取决于收益率变动的方向**。同样是放量，收益率上升时代表资金追逐（投资者愿意以更高的收益率买入），收益率下降时代表抛售压力（投资者急于卖出导致价格下跌）。

#### 2.2.3 参数选择

| 参数 | 默认值 | 选择依据 |
|------|--------|---------|
| 窗口大小 | 20个交易日 | 约等于1个交易月，平衡灵敏度与稳定性 |
| 收益率权重 | 0.5 | 使收益率贡献与成交金额贡献具有可比性 |

## 3. 数据来源与处理

### 3.1 数据来源说明

| 数据类型 | 来源 | 更新频率 | 用途 |
|---------|------|---------|------|
| 10年国债收益率 | Wind/金融终端 | 每日 | 计算收益率增长率，确定方向系数 |
| 债券成交金额 | Wind/金融终端 | 每日 | 计算成交金额增长率，反映市场活跃度 |

与MOVE指数需要2年、5年、10年、30年多个期限的期权数据相比，本研究仅需要两个现货数据源，大大降低了数据获取门槛。

### 3.2 数据处理要点

**滚动窗口计算**：采用20个交易日的滚动窗口，每个交易日基于过去20个交易日的数据计算增长率。这种方法既能捕捉短期趋势，又能过滤日度噪音。

**方向系数判定**：在每个计算时点，首先判断收益率在窗口期内的变动方向，然后据此设置方向系数。这一步骤是区分市场情绪类型的关键。

**幂等性设计**：采用UPSERT策略写入数据库，确保同一交易日的重复计算不会产生重复记录，支持增量更新和全量计算的灵活切换。

## 4. 核心分析框架

### 4.1 分析维度

从投研角度，本情绪指标框架包含以下分析维度：

**维度一：市场活跃度（成交金额增长率）**

成交金额的变化反映市场参与程度。高成交金额通常意味着：
- 市场关注度高，重要信息正在被定价
- 流动性充裕，大额交易可以顺利执行
- 可能存在较大的多空分歧

**维度二：价格趋势（收益率增长率）**

收益率的变动方向和幅度反映市场对基本面和政策预期的变化：
- 收益率上升：可能反映通胀预期上升、货币收紧预期或经济复苏预期
- 收益率下降：可能反映避险情绪上升、货币宽松预期或经济衰退预期

**维度三：情绪方向（方向系数的综合效应）**

方向系数将上述两个维度有机结合，输出单一的情绪信号：
- 正值：市场情绪乐观，买方力量占优
- 负值：市场情绪悲观，卖方力量占优
- 接近零：市场情绪中性或量价背离

### 4.2 信号解读指南

| 情绪指标值 | 市场状态 | 投资启示 |
|-----------|---------|---------|
| 显著为正（如>0.05） | 乐观放量 | 市场情绪积极，可能处于收益率上行周期的早期或中期，投资者追涨意愿强 |
| 显著为负（如<-0.05） | 恐慌抛售 | 市场情绪悲观，可能存在避险需求或流动性压力，需警惕行情加速下跌 |
| 接近零 | 中性/背离 | 市场情绪不明确，或存在量价背离现象，需结合其他指标综合判断 |

### 4.3 典型案例分析

为了更直观地理解指标的工作原理，我们分析两组典型的市场情景：

**情景A：乐观放量**

假设市场条件如下：
- 近20个交易日10年国债收益率从2.50%上升至2.55%（增长率+2%）
- 同期成交金额增加10%
- 收益率权重0.5

计算过程：
- 收益率增长率贡献：0.5 × 0.02 = 0.01
- 方向系数：+1（收益率上升）
- 成交金额贡献：0.1 × 1 = 0.1
- 情绪指标：0.01 + 0.1 = **0.11**

解读：情绪指标为正，表明市场处于乐观状态。收益率上升伴随成交放量，说明投资者愿意以更高的收益率买入债券，追逐收益的心态明显。

**情景B：恐慌抛售**

假设市场条件如下：
- 近20个交易日10年国债收益率从2.55%下降至2.50%（增长率-2%）
- 同期成交金额同样增加10%
- 收益率权重0.5

计算过程：
- 收益率增长率贡献：0.5 × (-0.02) = -0.01
- 方向系数：-1（收益率下降）
- 成交金额贡献：0.1 × (-1) = -0.1
- 情绪指标：-0.01 + (-0.1) = **-0.09**

解读：情绪指标为负，表明市场处于恐慌状态。虽然成交金额同样增加10%，但收益率下降意味着价格下跌，成交量的放大反映的是抛售压力而非追涨需求。

**关键洞察**：相同的成交金额变化（10%增长），在不同收益率方向下产生了截然相反的情绪信号（+0.11 vs -0.09）。这正是方向系数设计的核心价值——区分"谁在行动"以及"为何行动"。

## 5. 心得要点

### 要点一：方向性量价分析——从"发生了什么"到"意味着什么"

**背景**：在债市分析中，我们经常观察到成交量放大的现象。但传统分析往往停留在"成交量增加了"这一事实层面，较少深入追问"这次放量意味着什么"。

**做法**：引入方向系数，将成交金额增长率的贡献根据收益率变动方向进行调整。当收益率上升时，成交增加被视为乐观信号（投资者追涨）；当收益率下降时，成交增加被视为悲观信号（投资者出逃）。

**洞察**：债市情绪分析的核心不在于单纯的成交量变化，而在于成交量变化背后的驱动因素。方向系数的设计实际上是在回答一个根本性问题：**"这次放量是买家在追涨，还是卖家在出逃？"**

这一思路源于股市量价分析的启发，但针对债市特点进行了关键调整。在股市中，放量上涨通常被视为积极信号（买方力量强），放量下跌被视为消极信号（卖方压力大）。债市同理，但需要将"价格"转换为"收益率"来理解——收益率下降对应价格上涨，收益率上升对应价格下跌。

**关联**：这一心得与行为金融学中的"处置效应"概念相关——投资者倾向于过早卖出盈利资产、过久持有亏损资产。方向系数的负值信号可能部分反映这种心理机制在债市的表现。

### 要点二：现货数据构建情绪指标——在约束条件下创新

**背景**：国际主流的债市波动指标（如MOVE指数）依赖期权市场的隐含波动率数据。MOVE指数需要2年、5年、10年、30年四个期限的期权数据，通过加权平均计算隐含波动率。然而，中国债券期权市场相对不发达，数据可得性受限，这给情绪指标的构建带来了挑战。

**做法**：完全基于现货市场数据（成交金额和收益率）构建情绪指标。不依赖衍生品数据，而是通过增长率计算和方向调整来捕捉市场情绪。核心创新在于将"量"和"价"两个维度的信息有机融合，通过方向系数实现语义上的情绪判断。

**洞察**：在衍生品市场不发达或数据不可得的情况下，可以通过创造性的组合现有现货数据来构建有意义的情绪指标。关键在于深入理解市场运作逻辑，找到能够反映情绪的数据组合方式。

这一经验具有普适性。在新兴市场研究中，我们经常面临数据约束。与其等待完美的数据环境，不如思考：**"在现有数据条件下，我能做什么？哪些数据组合可能捕捉到我要研究的变量？"**

**关联**：这一思路与ESMA关于债券市场流动性的学术研究相关。那项研究使用主成分分析（PCA）方法聚合多个流动性指标，同样体现了"多指标融合"的思想。不同之处在于，本研究不是通过统计方法降维，而是通过经济学逻辑构建合成指标。

### 要点三：滚动窗口参数化——让假设显式化

**背景**：情绪指标计算需要平衡灵敏度与稳定性。窗口太短（如5个交易日）容易受日度噪音干扰，产生大量虚假信号；窗口太长（如60个交易日）则反应滞后，可能错过重要的情绪转折点。

**做法**：采用20个交易日（约1个月）作为默认窗口。这一选择基于以下考量：一个月的时间跨度足以过滤短期噪音，又不会过度平滑掉有意义的情绪变化。更重要的是，系统采用配置类和环境变量实现参数可调，支持不同分析场景的灵活配置。

**洞察**：参数化设计不仅提高了代码的复用性，更重要的是**将核心假设显式化**。在量化研究中，窗口大小、权重分配等参数往往是隐含的方法论假设。好的量化系统应该将这些假设"翻出来"，使其可见、可调、可测试。

这一实践为后续的参数敏感性分析预留了空间。例如，可以测试不同窗口大小（10天、20天、30天）下指标的稳定性，或者通过历史回测寻找最优参数组合。

**关联**：这一心得与量化投资中的"过拟合防范"理念相关。参数化使得我们可以明确知道模型中哪些是"假设"，哪些是"结论"，从而更好地理解模型的有效边界。

## 6. 实际应用指南

### 6.1 如何使用这项研究

**场景一：日常市场监控**

使用增量更新模式每日计算最新情绪指标：

1. 在每日收盘后运行增量更新程序
2. 系统自动获取近20个交易日数据，计算当日的增长率指标
3. 根据收益率变动方向确定方向系数
4. 计算当日情绪指标并写入数据库
5. 结合阈值设定预警机制（如情绪指标连续为负超过3天触发关注）

**场景二：历史行情回溯**

使用全量计算模式回溯历史情绪变化：

1. 运行全量计算程序，遍历所有历史数据
2. 系统为每个交易日计算对应的情绪指标
3. 结果写入数据库（UPSERT策略保证幂等性）
4. 可用于回测或分析历史行情中的情绪演变

**场景三：策略研究支持**

将情绪指标作为辅助变量纳入投资决策框架：

1. 观察情绪指标与债券收益率的相关性
2. 分析情绪极值（正/负）后的市场走势
3. 探索情绪指标对收益率拐点的预测能力
4. 结合其他技术指标构建综合信号

### 6.2 输出物说明

| 输出类型 | 内容 | 格式 | 用途 |
|---------|------|------|------|
| 情绪指标数值 | 每日sentiment_index | 数据库表 | 量化分析、回测 |
| 增长率明细 | yield_growth_rate, amount_growth_rate | 数据库表 | 诊断性分析 |
| 方向系数记录 | direction_factor | 数据库表 | 回溯验证 |

### 6.3 指标解读注意事项

在使用情绪指标时，需要注意以下几点：

1. **结合市场背景**：单一指标不足以支撑投资决策，应结合宏观经济、政策环境、资金面等因素综合判断。

2. **关注极值而非常态**：情绪指标在正常区间波动时参考意义有限，应重点关注极端值（如超过2个标准差）的出现。

3. **理解指标滞后性**：20日滚动窗口意味着指标对市场变化存在一定滞后，不宜用于高频交易信号。

4. **区分信号与噪音**：短期波动可能只是噪音，关注指标的持续性和趋势变化更为重要。

## 7. 延伸思考

### 7.1 可扩展的研究方向

**方向一：引入更多数据源构建综合情绪指数**

当前指标仅使用10年国债的成交金额和收益率数据。可以考虑引入：
- **回购利率**：反映资金面紧张程度，是债市情绪的重要驱动因素
- **信用利差**：反映风险偏好变化，信用利差收窄通常意味着风险情绪改善
- **期货持仓数据**：反映机构多空力量的对比
- **一级市场数据**：招标倍数、边际利率等反映配置需求

通过主成分分析（PCA）或加权平均方法，构建更全面的综合情绪指数。

**方向二：优化方向系数设计**
