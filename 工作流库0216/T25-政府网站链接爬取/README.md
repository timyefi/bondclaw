# T25 - 政府网站链接爬取

## 项目概述

本项目是一个政府网站链接爬取系统，用于自动化采集中国政府网站的公开数据。

### 主要功能

1. **网站发现与识别** - 自动发现和识别各级政府网站
2. **页面爬取与分析** - 爬取页面内容并进行智能分类
3. **内容分类与存储** - 对爬取的内容进行分类存储到数据库
4. **特征学习与优化** - 基于机器学习的页面类型识别

### 技术栈

- **爬虫引擎**: aiohttp (异步HTTP请求)
- **HTML解析**: BeautifulSoup + lxml
- **数据库**: MySQL (aiomysql + SQLAlchemy)
- **NLP处理**: jieba分词 + TF-IDF
- **机器学习**: scikit-learn (RandomForest)

## 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 配置环境变量

创建 `.env` 文件或设置环境变量：

```bash
# 数据库配置
DB_HOST=localhost
DB_PORT=3306
DB_NAME=site_dis
DB_USER=root
DB_PASSWORD=your_password

# 代理配置（可选）
PROXY_ENABLED=false
PROXY_URL=
```

### 3. 运行Notebook

打开 `重构.ipynb` 并运行所有单元格。

## 核心模块说明

### 1. ConfigManager - 配置管理

加载和管理YAML配置文件，支持动态配置更新。

### 2. DatabaseManager - 数据库管理

异步数据库操作，支持连接池和事务管理。

### 3. WebsiteManager - 网站管理

管理政府网站信息，包括省份映射和栏目类型识别。

### 4. PageAnalyzer - 页面分析

分析页面内容，提取标题、关键词，识别页面类型。

### 5. AsyncCrawler - 异步爬虫

并发爬取页面，支持请求限制和错误重试。

## 页面类型识别

系统支持识别以下页面类型：

| 类型 | 说明 | 关键词示例 |
|------|------|-----------|
| gov_info | 政府信息/政策文件 | 关于、通知、意见、办法 |
| notice | 公告通知 | 公告、通告、公示 |
| news | 新闻动态 | 新闻、动态、资讯 |
| guide | 办事指南 | 指南、流程、须知 |

## 注意事项

1. **遵守规则**: 爬取前请检查目标网站的 robots.txt
2. **合理并发**: 避免对目标服务器造成过大压力
3. **数据安全**: 不要硬编码敏感信息，使用环境变量
4. **合法合规**: 仅用于公开数据的学术研究和合法用途
