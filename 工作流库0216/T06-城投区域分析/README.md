# T06 - 城投区域分析

## 项目概述

城投区域分析项目专注于地方政府融资平台（城投债）的区域性信用风险分析和投资策略研究。通过对不同地区的城投平台进行深入分析，为投资者提供区域风险评级、利差分析和投资建议。

### 核心功能

1. **区域风险评级** - 建立区域信用风险评级体系
2. **利差分析** - 分析不同区域城投债的利差走势
3. **投资策略** - 提供区域配置建议和风险预警
4. **政策跟踪** - 跟踪地方债务政策变化

## 数据源

## 核心指标

### 区域信用评级指标

- 财政实力指标（30%）：一般公共预算收入、财政自给率
- 债务负担指标（30%）：债务率、负债率
- 经济活力指标（20%）：GDP增速、固定资产投资增速
- 金融支持指标（10%）：存款余额、贷款余额
- 政策环境评分（10%）

### 评级标准

| 评级 | 综合评分 |
|------|----------|
| AAA | >= 90 |
| AA+ | 80-89 |
| AA | 70-79 |
| AA- | 60-69 |
| A | < 60 |

## 注意事项

- 政策风险：关注地方债务政策变化
- 数据质量：确保数据的准确性和完整性
- 评级稳定性：避免评级频繁变动
- 风险控制：严格控制单一区域风险敞口
