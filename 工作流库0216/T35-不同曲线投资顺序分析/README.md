# T35 - 不同曲线投资顺序分析

## 项目概述

本项目专注于研究不同类型债券（国债、国开、农发等）的投资顺序和收益率曲线形态，分析不同品种之间的利差关系和投资时点选择，为跨品种投资策略提供参考。

## 功能特性

- **收益率曲线数据采集**：从数据库获取多种债券类型的收益率曲线数据
- **投资顺序分析**：分析不同债券类型在不同久期区间的投资顺序
- **多维度策略分析**：从品种维度、久期维度、资质维度三个维度分析投资策略
- **历史分位点计算**：计算债券利差的历史分位点，为投资决策提供参考
- **可视化分析**：生成策略对比图表和收益分析图

## 核心逻辑

### 数据流程

```
周期标注 -> 收益率数据 -> 策略生成 -> 周期分割 -> 收益计算 -> 统计汇总 -> 可视化
```

### 债券收益计算

```
总收益 = 票息收益 + 价格收益
票息收益 = 期初收益率 x 持有天数 / 365
价格收益 = -修正久期 x 收益率变动 x 持有天数 / 365
```

### 三维度策略

1. **品种维度**：利率债/金融债/信用债的顺序组合
2. **久期维度**：长久期优先 vs 短久期优先
3. **资质维度**：高资质优先 vs 弱资质优先

## 配置参数

主要配置参数在 `config.py` 中定义：

- `TYPE_MAPPING`: 债券类型映射
- `RATING_MAPPING`: 评级映射
- `DURATION_MAP`: 期限-久期映射
- `YIELD_DOWN_CYCLES`: 历史收益率下行周期定义

## 输出文件

- `收益率下行周期标注.csv`：周期标注数据
- `债券收益率数据.csv`：债券收益率数据
