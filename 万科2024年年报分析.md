# 万科2024年年报分析案例

## 基本信息
- **公司**: 万科企业股份有限公司
- **报告期**: 2024年度
- **报告类型**: 交易商协会简版年报（银行间债券市场信息披露）
- **文件大小**: ~5.5MB
- **分析日期**: 2026-03-13

## 关键发现

### 1. 报告类型确认
5.5MB的年报仍然是交易商协会简版年报！文件大小阈值需进一步上调。
**学习点**: 简版年报的文件大小可达5.5MB，阈值应提高到至少10MB。包含较多表格和文字的简版报告也可以有较大的文件体积。

### 2. 重大风险信号 — 大额亏损
- **合并报表范围亏损超过上年末净资产10%**: 是
- 公司报告期出现大额亏损
- **这是极为重要的行业信号**: 万科作为头部房企出现大额亏损，反映行业系统性风险

### 3. 对外担保情况（详细数据）
| 担保类型 | 金额（亿元） |
|----------|-------------|
| 对外担保（不含子公司） | 18.43 |
| 对子公司担保 | 237.42 |
| 子公司对子公司担保 | 486.65 |
| **合计** | **742.50** |
| 占归母净资产比重 | 36.62% |

**分析**: 万科担保结构以子公司间互保为主（486.65亿），反映房企项目公司模式下的内部担保链条。对外净担保（18.43亿）比例极低，说明对外风险敞口有限。

### 4. 合并报表范围重大变动
- 新增221家子公司，减少205家子公司
- 大幅度的子公司增减反映房地产项目公司的频繁设立和清算

- 审计意见: 标准无保留

## 技能进化记录
1. **再次修正文件大小阈值**: 简版年报可达5.5MB，阈值应提高到10MB
2. **新增大额亏损关注**: 合并报表亏损超净资产10%是重要风险信号
3. **新增担保结构分析**: 区分"对外担保"vs"对子公司担保"vs"子公司间互保"
4. **新增子公司变动规模分析**: 频繁大量增减子公司是房企项目公司模式的特征
