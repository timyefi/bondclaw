## 一、项目概述与投资背景

### 1.1 非税收入在固收投研中的定位

在地方政府债券和城投债投资分析中，财政收入质量是评估区域信用风险的核心维度之一。一般公共预算收入由税收收入和非税收入两部分构成，其中非税收入主要包括行政事业性收费、罚没收入、国有资源（资产）有偿使用收入、国有资本经营收入等。

与税收收入的稳定性和可预期性不同，非税收入具有明显的政策依赖性和一次性特征。当地方政府面临税收增长乏力时，往往通过盘活国有资产、增加行政收费等方式补充财力。因此，非税收入占比成为衡量地方财政健康程度的重要反向指标：占比越高，意味着财政质量越差、收入稳定性越弱。

### 1.2 当前地方财政环境分析

根据财政部最新公布的数据，2024年全国一般公共预算非税收入达4.47万亿元，同比增长25.4%，增速显著高于税收收入的-3.4%。这一分化反映了地方政府在土地出让收入大幅下滑背景下，通过多渠道筹集财政资金的现实压力。

进入2025年，非税收入增速明显放缓。上半年全国非税收入2.27万亿元，同比增长3.7%，增幅较一季度回落5.1个百分点。5月、6月非税收入甚至出现单月同比下降2.2%和3.7%的情况，显示非税收入的可持续性面临挑战。

从债务负担角度看，地方债务利息占广义财政收入比重已接近20%。在财政收支紧平衡状态下，地方政府偿债压力逐年上升，部分区域的城投平台已暴露出明显的违约风险。

### 1.3 本项目的投研价值

本项目建立了一套系统化的非税收入比例计算与追踪框架，通过省级层面的时间序列分析，为城投债区域筛选和信用风险监控提供量化支持。具体而言，该框架能够：

- **横向比较**：识别非税收入占比异常偏高的省份，提示潜在财政质量风险
- **纵向追踪**：监控各省份非税收入比例的时间变化趋势，捕捉财政质量恶化信号
- **信号预警**：当某省份非税收入占比突破阈值（如35%）或出现快速上升时，触发深度分析需求

## 二、数据处理架构与核心算法

### 2.1 数据源与输入结构

项目使用的数据源为"一般公共预算收入累计值等.xlsx"，该文件包含全国各省份的月度财政数据。数据结构采用宽表格式，每个省份包含三列数据：一般公共预算收入、税收收入、非税收入。

原始数据的列名格式为"省份:指标类型"，例如"北京:一般公共预算收入"、"北京:税收收入"、"北京:非税收入"。这种命名方式便于数据解析，但也要求处理逻辑具备一定的容错能力。

### 2.2 核心计算逻辑解析

项目的核心函数`calculate_non_tax_ratio`实现了非税收入比例的计算，其算法设计体现了投研数据处理的三个重要原则：

**原则一：数据韧性与多重保障**

函数首先尝试使用直接计算法：
```
非税收入比例 = 非税收入 / 一般公共预算收入
```

当某省份的非税收入数据缺失时，自动切换为间接计算法：
```
非税收入比例 = 1 - 税收收入 / 一般公共预算收入
```

这种双重逻辑确保了在数据源不完整时仍能产出有效指标。在地方财政数据披露实践中，不同省份的披露口径和及时性存在差异，这种容错设计保证了时间序列分析的连续性。

**原则二：省份去重与数据整合**

由于原始数据中每个省份可能对应多列（一般公共预算收入、税收收入、非税收入），函数通过`processed_provinces`集合记录已处理的省份，避免重复计算。同时，通过字符串匹配识别属于同一省份的不同指标列，确保数据正确关联。

**原则三：比例计算的数学一致性**

无论采用直接法还是间接法，最终产出的非税收入比例在数学上是等价的：
```
非税收入比例 = 非税收入 / 一般公共预算收入
             = (一般公共预算收入 - 税收收入) / 一般公共预算收入
             = 1 - 税收收入 / 一般公共预算收入
```

这种一致性保证了跨省份、跨时间可比性。

### 2.3 数据清洗与异常值处理

在计算出原始非税收入比例后，项目执行了以下数据清洗步骤：

**步骤一：数值类型转换**

将所有比例列转换为数值类型，非数值数据自动转为空值。这一步骤处理了原始数据中可能存在的文本型数值或特殊标记（如"-"表示数据缺失）。

**步骤二：负值替换为空值**

```python
non_tax_ratio_df.loc[non_tax_ratio_df[col] < 0, col] = None
```

非税收入比例为负值在实际经济意义上是不合理的，其出现通常源于以下情况：
- 数据口径调整导致的暂时性异常
- 会计处理中的特殊项目冲抵
- 数据录入或传输错误

将负值替换为空值而非直接删除，保留了时间序列的完整性，便于后续填充处理。

**步骤三：前向填充缺失值**

```python
non_tax_ratio_df.ffill(inplace=True)
```

前向填充假设财政结构短期内保持稳定，这是财政分析中的常见假设。地方政府财政结构通常不会发生突发性剧变，使用上一期的有效值填充当前缺失值具有合理性。

### 2.4 数据存储与输出

处理完成的数据通过SQLAlchemy写入MySQL数据库，表名为`non_tax_ratio_df`。同时，项目生成以下输出文件：

| 输出文件 | 内容 | 用途 |
|---------|------|-----|
| non_tax_ratio_trend.png | 多省份非税收入比例时间序列图 | 趋势可视化分析 |
| non_tax_ratio_heatmap.png | 各省份最近24个月热力图 | 区域比较与异常识别 |
| non_tax_ratio_processed.xlsx | 清洗后的非税收入比例数据 | 下游分析数据源 |

## 三、投研应用场景与方法论

### 3.1 城投债信用分析中的应用

在城投债信用分析框架中，非税收入比例可作为区域财政质量的代理指标。具体应用包括：

**应用一：区域准入筛选**

在进行城投债投资时，首先可设置非税收入比例的阈值筛选。根据市场实践，非税收入占比超过35%通常被视为财政质量较差的信号。对于超过阈值的区域，需要更加谨慎地评估其偿债能力，或要求更高的信用利差补偿。

**应用二：持仓监控预警**

对于已持仓的城投债，定期跟踪发行人所在省份的非税收入比例变化。若出现以下情况，需触发深度分析或减仓考虑：
- 非税收入比例连续多个季度上升
- 非税收入比例在短期内大幅跳升（如季度环比上升超过5个百分点）
- 非税收入比例突破历史阈值

**应用三：同业比较与相对价值**

在同一信用评级的不同区域之间，非税收入比例可作为区分财政质量的细化指标。在其他条件相近的情况下，优先选择非税收入比例较低、税收收入占比较高的区域，因其财政收入更具稳定性和可持续性。

### 3.2 时间序列分析的方法论

本项目生成的非税收入比例时间序列数据支持以下分析维度：

**趋势分析**

通过观察非税收入比例的长期趋势，判断地方财政质量是改善还是恶化。上升趋势可能反映以下情况：
- 税收增长乏力，地方政府被迫寻求非税收入补充
- 国有资产处置力度加大，可能是一次性行为
- 行政事业性收费增加，反映财政压力

**周期性分析**

非税收入比例可能存在一定的季节性或周期性特征。例如，部分省份可能在年末集中处置国有资产，导致非税收入比例在特定月份出现峰值。识别这些周期性特征有助于区分趋势性变化和季节性波动。

**结构性断点检测**

当非税收入比例出现跳跃式变化时，需要进一步分析其原因。可能是政策调整（如收费政策变化）、经济冲击（如疫情影响）或数据口径调整。结构性断点的识别对于判断财政质量的真实变化至关重要。

### 3.3 可视化分析的投研解读

**时间序列图的解读要点**

项目生成的多省份非税收入比例时间序列图支持以下分析：

1. **水平比较**：不同省份在同一时间点的非税收入比例差异，反映财政质量的高低
2. **趋势比较**：不同省份的非税收入比例变化方向是否一致，判断是全国性因素还是区域性因素主导
3. **收敛性分析**：观察各省份非税收入比例是否趋于收敛，反映财政质量的均衡化程度

**热力图的解读要点**

热力图聚焦最近24个月的数据，采用红黄绿渐变色（`RdYlGn_r`）表示非税收入比例的高低。红色代表高比例（财政质量较差），绿色代表低比例（财政质量较好）。

热力图的横向（时间轴）分析可以识别趋势变化，纵向（省份轴）分析可以识别异常区域。颜色深浅的突变点往往是需要关注的信号点。

## 四、心得要点与深度洞察

### 4.1 心得一：数据韧性与连续性保障是投研系统的基石

在分析本项目的代码架构时，最令人印象深刻的是其数据处理逻辑的韧性设计。`calculate_non_tax_ratio`函数采用的双重计算法并非简单的代码冗余，而是深刻理解了地方财政数据披露实际困难后的工程化解决方案。

**现实背景**

地方财政数据的披露存在以下挑战：
- 不同省份披露口径存在差异，部分省份可能未单独披露非税收入数据
- 数据更新及时性不一致，部分省份可能存在延迟披露
- 历史数据可能存在追溯调整，导致口径变化

**设计思想**

通过提供直接法和间接法两种计算路径，系统在数据完整性方面实现了"或"逻辑而非"与"逻辑。只要任一计算路径的数据可用，就能产出有效指标。这种设计思想在投研系统架构中具有普遍借鉴意义：当核心指标的计算依赖多个数据源时，应优先设计降级方案而非依赖单一数据源。

**延伸思考**

在实际应用中，可进一步扩展这种韧性设计：
- 当税收收入和非税收入数据同时缺失时，考虑使用上期比例或同类省份均值进行估算
- 建立数据质量评分机制，根据数据来源（直接计算vs间接计算）给予不同的置信度权重
- 对于数据缺失较严重的省份，在分析报告中给予标注提示

### 4.2 心得二：短期风险监控导向体现了固收投资的时效性需求

项目中热力图特意聚焦最近24个月的数据，而非展示全部历史数据。这一看似简单的设计选择，实际上深刻反映了固收投资的时效性需求。

**投资视角的时间窗口**

城投债的久期通常在3-5年，投资者更关注中短期内的财政质量变化。24个月的窗口期恰好覆盖约2年的观察期，能够有效捕捉：
- 地方政府财政政策的边际变化
- 经济周期波动对财政的影响
- 突发事件（如疫情、自然灾害）的冲击

**历史数据的局限性**

过度依赖历史数据可能产生误导。例如，某省份5年前的非税收入比例可能很低，但近年来因土地财政退潮而大幅上升。如果平均看待历史数据，可能低估当前的财政压力。

**监控导向的实用主义**

在投研实践中，完整的学术分析固然重要，但投资决策往往需要在有限时间内完成。聚焦最近24个月的热力图提供了一种高效的信息概览方式，分析师可以快速识别需要深入研究的异常区域。

**延伸思考**

24个月窗口期可根据具体应用场景调整：
- 对于短期限债券（1-2年），可缩短至12个月以更加敏感
- 对于长期限债券（5年以上），可延长至36个月以捕捉更完整的周期
- 可考虑设置多档窗口期（12个月、24个月、36个月）并行展示，提供不同时间尺度的视角

### 4.3 心得三：财政指标的数据清洗需尊重经济逻辑

项目中关于负值替换和前向填充的处理逻辑，体现了对财政指标本质特征的深刻理解。与一般的时间序列数据不同，财政指标具有其特殊的经济含义和统计特性。

**负值的经济学解读**

非税收入比例为负值在经济学上是不合理的：非税收入和一般公共预算收入均为正值，其比值不可能为负。因此，负值的出现必然是数据问题而非经济现象。

将负值替换为空值而非直接删除或修正为0，是基于以下考量：
- 删除会导致时间序列中断，影响连续性分析
- 修正为0会引入虚假信息，可能误导分析
- 替换为空值保留了数据的原始状态，可通过后续填充处理

**前向填充的经济学合理性**

财政结构具有惯性特征：地方政府的收入来源构成在短期内通常保持稳定，不会发生突发性剧变。这一特性使得前向填充在财政数据分析中具有合理性。

相比之下，如果分析的是股票收益率等高频金融数据，前向填充显然是不合适的，因为金融数据具有较强的波动性和随机性。

**数据清洗的通用原则**

从本项目的数据清洗逻辑中，可以提炼出适用于更广泛投研场景的原则：
1. **理解数据的经济含义**：在处理数据之前，首先要理解指标的经济含义和合理取值范围
2. **区分信号与噪音**：异常值可能是真实的信号（需要深入分析）也可能是噪音（需要清洗），要根据具体情境判断
3. **保留处理痕迹**：在数据清洗过程中保留原始数据和中间结果，便于追溯和验证

## 五、延伸思考与改进方向

### 5.1 指标局限性分析

非税收入比例作为财政质量的代理指标，存在以下局限性：

**局限性一：非税收入的内部结构差异**

非税收入包含多种类型，其可持续性和质量存在差异：
- 国有资源（资产）有偿使用收入：往往是一次性的资产处置，不可持续
- 行政事业性收费：相对稳定，但过度依赖可能抑制经济活力
- 罚没收入：波动性大，且与执法力度相关

仅仅关注非税收入总量比例，可能忽略其内部结构变化。建议在数据条件允许时，进一步分解非税收入的各组成部分。

**局限性二：省级数据的颗粒度不足**

本项目的分析基于省级数据，而城投债发行主体通常在地级市或区县级。省级非税收入比例可能掩盖省内不同区域的财政质量差异。例如，某省份整体非税收入比例较低，但省内欠发达地市的非税收入比例可能很高。

建议在有条件时，收集地级市层面的财政数据，进行更精细化的分析。

**局限性三：时间滞后性**

财政数据的披露存在一定的滞后性，月度数据通常在次月中下旬公布。这意味着分析结果反映的是滞后约一个月的财政状况。在财政质量快速恶化的情况下，可能存在信号延迟。

### 5.2 与其他财政指标的联动分析

非税收入比例应与其他财政指标联动分析，形成更完整的财政质量评估框架：

**联动指标一：财政自给率**

财政自给率 = 一般公共预算收入 / 一般公共预算支出

该指标反映地方政府的财政独立性。非税收入比例高且财政自给率低的区域，财政风险最高。

**联动指标二：政府性基金收入占比**

政府性基金收入主要来源于土地出让收入。在土地财政退潮背景下，政府性基金收入占比下降的区域可能面临更大的财政压力，也更有可能转向非税收入补充财力。

**联动指标三：债务率**

债务率 = 政府债务余额 / 综合财力

非税收入比例上升往往伴随着财政压力加大，此时债务率的边际变化尤为重要。

### 5.3 模型扩展方向

基于当前的分析框架，可以考虑以下模型扩展方向：

**扩展一：构建财政质量综合评分模型**

将非税收入比例、财政自给率、税收收入增速、债务率等指标纳入综合评分体系，形成量化的财政质量评分。该评分可作为城投债内部评级的输入因子。

**扩展二：建立异常检测模型**

基于历史数据建立非税收入比例的动态阈值模型。当某省份的非税收入比例突破动态阈值时，自动触发预警。可考虑使用统计方法（如3倍标准差）或机器学习方法（如孤立森林）进行异常检测。

**扩展三：开发财政压力指数**

将非税收入比例的变化率与其他财政指标的变化率进行加权合成，构建地方财政压力指数。该指数可用于监控全国各省份的财政压力动态，为投资决策提供前瞻性指引。
