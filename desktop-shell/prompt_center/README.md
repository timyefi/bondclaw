# 模板中心面板

这一层描述桌面壳里的“模板中心”页面模型。

## 组成

- `prompt_center_model.py`：把角色模板、样例、数量和快捷动作组合成可渲染面板。

## 设计目标

- 默认展示每个角色的模板数量和样例数量。
- 允许按角色浏览模板卡。
- 允许查看某个角色的默认模板。
- 允许从模板中心跳到具体模板卡。
- 允许在首页快速看到默认角色与热门角色卡。
- 保持本地目录读取优先，不绑定复杂后端。

## 入口

桌面壳可以通过 `desktop-shell/launch_bondclaw.py --prompt-center-panel` 查看该面板的结构化数据。
