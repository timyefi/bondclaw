# T40 - 区域基本面

## 项目概述

区域基本面项目专注于分析不同区域的经济基本面，包括GDP增速、财政收入、产业结构、人口结构、投资情况等，为区域债投资和区域风险评估提供基本面数据支持。

## 核心功能

1. **区域经济数据采集与处理** - 从数据库获取区域基本面数据
2. **综合财力计算** - 计算区域综合财力和财政自给率
3. **经济活力评估** - 评估GDP增速、产业升级度、投资活跃度
4. **发展潜力评估** - 评估人口红利、城镇化红利、增长惯性
5. **区域综合评分** - 财政40% + 经济30% + 潜力30%
6. **风险评级** - AAA/AA+/AA/AA-/A/BBB/BB 七级评级
7. **区域对比分析** - 生成各维度排名

## 数据源

- 主数据表: `bond.fiscal_region_city_county`
- 字段说明:
  - `DT` - 报告日期
  - `省` - 省份
  - `市` - 城市
  - `区县` - 区县
  - `地区级别` - 省级/地市级/区县级
  - `城投平台有息债务余额` - 城投债务数据
  - `城投债券余额` - 城投债券数据
  - `地方债务余额` - 地方债务数据

## 核心算法

### 综合评分公式

```
综合评分 = 财政实力评分 * 0.4 + 经济活力评分 * 0.3 + 发展潜力评分 * 0.3
```

### 风险评级标准

| 评级 | 综合评分 | 负债率上限 |
|------|----------|------------|
| AAA  | >= 80    | < 1.0      |
| AA+  | >= 70    | < 1.5      |
| AA   | >= 60    | < 2.0      |
| AA-  | >= 50    | < 3.0      |
| A    | >= 40    | -          |
| BBB  | >= 30    | -          |
| BB   | < 30     | -          |

## 债务数据修正逻辑

1. **债务余额 < 债券余额** -> 用债券余额填充
2. **市本级数据修正** -> 等于下辖区县债务总和
3. **省市级数据修正** -> 等于下级区县债务总和
4. **缺失数据填充** -> 使用相邻日期数据或地方债务余额的2倍估算
