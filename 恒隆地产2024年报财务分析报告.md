# 恒隆地产有限公司 2024年度财务分析报告

**报告日期：** 2026年3月14日
**分析对象：** 恒隆地产2024年年报（港股HKFRS准则）
**分析师：** Claude Code Financial Analyzer

## 一、执行摘要

### 1.1 核心结论

恒隆地产作为**纯商业地产运营商**，2024年业绩表现稳健但面临挑战：

| 维度 | 评估 | 关键指标 |
|------|------|----------|
| **盈利能力** | ⭐⭐⭐ | 收入112.42亿港元(+9%)，归母净利22.2亿港元(-21%) |
| **偿债能力** | ⭐⭐⭐⭐⭐ | 现金短债比1.09x，净负债率21.2%，利息保障倍数4.05x |
| **资产质量** | ⭐⭐⭐⭐ | 投资物业1,665亿港元（-1.49%），公允价值小幅下降 |
| **现金流** | ⭐⭐⭐⭐ | 现金103.03亿港元(+92.5%)，未动用信贷125.35亿港元 |
| **风险水平** | ⭐⭐⭐⭐ | 短期债务激增110%，但流动性充足 |

### 1.2 关键发现

1. **港股特有会计政策**：采用HKFRS准则，投资物业按**公允价值计量**，利润波动较大
2. **利息100%资本化**：11.83亿港元利息全部资本化，实际利率4.2%
3. **纯租赁商业模式**：租金收入占比约92%，物业销售收入仅1.89亿港元
4. **短期债务压力**：一年内到期债务93.40亿港元，同比增长110.65%
5. **利率对冲策略**：定息债务41%，浮息59%，使用利率掉期对冲风险

### 4.2 风险详细分析

#### 4.2.1 流动性风险

**风险点：**
- 2024年一年内到期债务93.40亿港元，同比+110.65%
- 占总有息债务比例从8.7%上升至16.2%

**缓释因素：**
- 现金及现金等价物101.98亿港元，现金短债比1.09x
- 未动用银行信贷额度125.35亿港元
- 中期票据计划剩余可发行20.78亿美元

**结论：** 短期偿债能力充足，但需关注2025年再融资情况

#### 4.2.2 利率风险

**风险点：**
- 浮息债务336.51亿港元，占比58.6%
- 若利率上升100bp，年净利息支出增加2.35亿港元

**缓释因素：**
- 利率掉期合约对冲237.25亿港元（41%）
- 美联储降息周期有利于降低融资成本
- 债券利率较低（2.0%-5.0%），期限较长

**结论：** 利率风险可控，降息周期有利

#### 4.2.3 汇率风险

**风险点：**
- 内地投资物业1,040亿港元（占比67%），以人民币计价
