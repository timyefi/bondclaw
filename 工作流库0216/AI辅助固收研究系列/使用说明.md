# AI辅助固收研究系列报告 - 使用说明

## 概述

本系统用于自动生成66篇AI辅助固收研究系列报告，每篇报告以对应项目（T01-T66）为案例进行详细讲解。

**核心特点**：
- ✅ 完全自动化执行，无需人工干预
- ✅ 三子代理流水线架构，解决上下文限制
- ✅ 聚焦项目独特性，避免泛泛而谈
- ✅ 支持会话中断后恢复

## 快速开始

### 1. 启动全量自动执行

在Claude Code中输入以下指令：

```
启动AI辅助固收研究系列报告生成任务，批量生成所有66篇报告，全自动执行，不需要中途确认。
```

然后将`prompts/主控Prompt.md`的内容提供给Claude Code。

### 2. 批量执行指定数量

```
启动AI辅助固收研究系列报告生成任务，批量生成10篇报告，全自动执行。
```

### 3. 从中断处恢复

如果执行过程中会话意外中断，重新输入：

```
恢复AI辅助固收研究系列报告生成任务，从中断位置继续批量生成剩余报告。
```

## 执行流程

### 三子代理流水线

每篇报告的生成分为三个阶段，由独立的子代理完成：

```
┌─────────────────────────────────────────────────────────────┐
│  探索子代理                   分析子代理                   撰写子代理
│       │                           │                           │
│       ▼                           ▼                           ▼
│  扫描项目结构           读取探索结果.json           读取分析结果.json
│  浏览核心代码           深度代码分析               生成最终报告
│  网络搜索补充           心得要点提炼               输出.md文件
│       │                           │                           │
│       ▼                           ▼                           ▼
│  探索结果.json          分析结果.json             第N篇-TXX-xxx.md
└─────────────────────────────────────────────────────────────┘
```

### 自动化执行循环

主控会话自动执行以下循环：

1. 读取进度文件，获取下一个待处理项目
2. 检查是否达到批量限制
3. 依次启动三个子代理
4. 更新进度文件
5. 输出完成摘要
6. 自动继续下一个项目（无需确认）

## 进度追踪

### 查看进度

```
显示当前报告生成进度。
```

### 进度文件格式

```json
{
  "statistics": {
    "completed": 10,    // 已完成数量
    "pending": 55,      // 待处理数量
    "skipped": 1,       // 跳过数量
    "errors": 0         // 错误数量
  }
}
```

### 状态说明

| 状态 | 含义 |
|------|------|
| pending | 等待处理 |
| in_progress | 正在处理 |
| completed | 已完成 |
| skipped | 已跳过 |
| error | 处理失败 |

## 注意事项

### ⚠️ 禁止清单

报告内容应避免：
- ❌ "本项目使用Python进行数据处理"
- ❌ "实现了数据的自动化采集"（没有具体说明）
- ❌ "提高了研究效率"（没有具体说明）
- ❌ 编造具体数据（如"效率提升5倍"）

### ✅ 推荐做法

- 聚焦项目独特性
- 包含心得要点章节
- 使用业务语言
- 提供具体应用场景

## 常见问题

### Q: 执行过程中会话中断怎么办？

A: 重新发送恢复指令，系统会自动从中断位置继续。

### Q: 如何只生成单个项目的报告？

A: 使用指令：`生成T01-每日数据获取任务的报告。`

### Q: 报告生成后存放在哪里？

A: 报告存放在 `reports/` 目录下，文件名格式为 `第N篇-TXX-项目名称.md`。

### Q: 如何修改批量执行数量？

A: 在启动指令中指定，如：`批量生成5篇报告。`
