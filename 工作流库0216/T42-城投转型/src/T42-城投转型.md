## 核心业务逻辑

### 1. 变更类型分类

#### "退出" - 退出融资平台
- 城投融资平台完全退出政府融资职能
- 转型为市场化运作的国有企业
- 不再承担政府融资功能

#### "市场化转型" - 市场化转型
- 按照市场化要求建立公司法人治理结构
- 按照《公司法》、《公司章程》规定开展市场化融资
- 存量债务不属政府债务，不存在替政府举债情形

#### "隐性债务清零" - 隐性债务清零
- 隐性债务已清零
- 退出融资平台监管范畴
- 可能仍有经营性金融债务

---

### 2. 变更原因详解

#### 2.1 市场化转型
**标准表述：**
```
为深入贯彻落实中央、省关于地方政府性债务管理要求，
进一步理清政府与企业边界，
推动融资平台公司市场化转型，
防范和化解债务风险，
[公司名称]已转型为市场化运作的国有企业。
经研究，决定从即日起退出政府融资平台，
不再承担政府融资职能。

转型后的国有企业依法开展市场化自主经营、
自负盈亏、自我约束、自我发展，承担有限责任。
```

**关键要素：**
- 深入贯彻落实债务管理要求
- 理清政府与企业边界
- 推动融资平台公司市场化转型
- 防范和化解债务风险
- 自即日起退出政府融资平台
- 不再承担政府融资职能
- 市场化自主经营、自负盈亏
- 承担有限责任

---

#### 2.2 隐性债务清零
**标准表述：**
```
[公司名称]退出融资平台名单，
实现隐性债务和融资平台双清零。
```

**关键要素：**
- 退出融资平台名单
- 隐性债务清零
- 融资平台双清零

---

#### 2.3 其他常见原因
- 不承担政府融资职能，不涉及新增地方政府隐性债务
- 其他应收款项主要系代垫购置土地款项，为经营性款项
- 主营业务市场化运作，不属于地方融资平台公司
- 非融资平台公司，上述资产不是公益性资产，不是储备土地

## 数据库表结构设计

### 表名：转型城投清单

```sql
CREATE TABLE 转型城投清单 (
    id BIGINT AUTO_INCREMENT PRIMARY KEY,
    itName VARCHAR(255) NOT NULL COMMENT '公司名称',
    itCode2 VARCHAR(255) NOT NULL COMMENT '公司代码2',
    regionName VARCHAR(255) NOT NULL COMMENT '区域名称',
    regionCode VARCHAR(50) NOT NULL COMMENT '区域代码',
    releaseDate DATE NOT NULL COMMENT '发布日期',
    changeType VARCHAR(50) NOT NULL COMMENT '变更类型',
    changeReason VARCHAR(255) COMMENT '变更原因',
    reasonDescription TEXT COMMENT '原因描述',
    publishEntity VARCHAR(50) NOT NULL COMMENT '发布主体',
    url VARCHAR(500) COMMENT '文件链接',
    guid VARCHAR(50) NOT NULL UNIQUE COMMENT '唯一标识',
    sourceFlag TINYINT NOT NULL COMMENT '来源标志(1=官方,2=企业)，
    subjectRating VARCHAR(10) COMMENT '主体评级',
    bondStockSize DECIMAL(20,2) COMMENT '债券库存规模',
    issueBond VARCHAR(10) COMMENT '是否发债',
    actualController VARCHAR(255) COMMENT '实际控制人',
    actualControllerCode VARCHAR(255) COMMENT '实际控制人代码',
    province VARCHAR(100) NOT NULL COMMENT '省份',
    city VARCHAR(100) NOT NULL COMMENT '城市',
    county VARCHAR(100) COMMENT '区县',
    description TEXT COMMENT '描述',
    trSimpleName VARCHAR(255) COMMENT '交易简称',
    trCode VARCHAR(255) COMMENT '交易代码',
    symbol VARCHAR(255) COMMENT '股票代码',
    createdAt TIMESTAMP DEFAULT CURRENT_TIMESTAMP COMMENT '创建时间',
    updatedAt TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP COMMENT '更新时间',
    
    INDEX idx_province (province),
    INDEX idx_city (city),
    INDEX idx_releaseDate (releaseDate),
    INDEX idx_changeType (changeType),
    INDEX idx_publishEntity (publishEntity)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COMMENT='城投融资平台转型清单';
```
