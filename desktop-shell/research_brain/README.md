# 信息中心面板

这一层描述桌面壳里的“信息中心”页面模型。

## 组成

- `research_brain_model.py`：把订阅源、案例卡、案例详情和筛选条件组合成可渲染面板。

## 设计目标

- 默认展示订阅源、案例数和推荐动作。
- 默认再展示来源卡和案例高亮，方便快速扫一眼当前信息源和重点案例。
- 默认再展示主题总览，方便快速扫一眼宏观、利率、信用、转债等研究重心。
- 允许按角色和主题浏览案例卡。
- 允许直接查看单条案例详情。
- 允许从案例详情直接读取对应 Prompt 快捷入口。
- 详情区内部按证据卡、Prompt 卡和操作卡三段式组织。
- 保持本地缓存优先，不让页面绑定复杂后端。

## 入口

桌面壳可以通过 `desktop-shell/launch_bondclaw.py --research-brain-panel` 查看该面板的结构化数据。
