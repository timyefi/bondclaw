# T43 - 理财监测系统

## 项目概述

理财监测是一个持续追踪理财市场表现的数据采集和分析系统，主要监控理财产品的业绩表现、规模变化、期限分布以及纯债基金的收益情况。

## 功能模块

### 1. 理财业绩跟踪
- **数据源**: 同花顺 iFinD p04838 接口
- **功能**: 获取理财产品业绩数据，计算年化收益率
- **输出**: `理财业绩跟踪` 表

### 2. 纯债基金业绩表现
- **数据源**: 数据库 fund.marketinfo + fund.basicinfo
- **功能**: 获取纯债基金净值数据，计算滚动收益率
- **输出**: `fund.纯债基金业绩表现` 表

### 3. 理财规模跟踪
- **数据源**: 证券之星网站爬虫
- **功能**: 爬取理财市场规模数据
- **输出**: `理财规模` 表

### 4. 理财期限跟踪
- **数据源**: 同花顺 iFinD p02186 接口
- **功能**: 获取理财产品期限分布数据
- **输出**: `理财期限跟踪` 表

### 核心算法

#### 年化收益率转换

原始数据是期间收益率，需要转换为年化收益率：

- **今年以来收益率**: `年化收益率 = 原始收益率 * (365 / 实际天数)`
- **近1月收益率**: `年化收益率 = 原始收益率 * (365 / 30)`
- **近3月收益率**: `年化收益率 = 原始收益率 * (365 / 90)`
- **近6月收益率**: `年化收益率 = 原始收益率 * (365 / 180)`
- **近1年收益率**: 无需转换（已经是年化）

#### 滚动收益率计算

```python
# 近1月年化收益率
(x / x.shift(30) - 1) * 365 / 30

# 近1年收益率
(x / x.shift(365) - 1)
```

## 数据流程

```
同花顺 p04838 --> 理财业绩数据 --> 年化转换 --> 理财业绩跟踪表
同花顺 p02186 --> 理财期限数据 --> 筛选合计 --> 理财期限跟踪表
证券之星网页 --> 爬取规模数据 --> 正则提取 --> 理财规模表
数据库净值 --> 填充日期 --> 滚动计算 --> 纯债基金业绩表
```

## 注意事项

1. **年化转换**: 今年以来收益率的天数计算从上一年12月31日开始
2. **数据填充**: 净值数据使用前向填充（ffill）处理缺失日期
3. **UPSERT策略**: 使用 ON DUPLICATE KEY UPDATE 避免重复数据
4. **排除基金**: 排除 511010.OF（国债ETF）等非纯债基金
