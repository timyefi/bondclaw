# T13 - 债券指数研究

## 项目概述

本项目是一个完整的债券指数分析系统，主要功能包括：

1. **数据获取**：从MySQL数据库获取债券指数数据和价格数据
2. **指标计算**：计算收益率、最大回撤、年化波动率等核心指标
3. **多周期分析**：支持30天、3个月、半年、1年等多个时间周期的分析
4. **分组统计**：按收益率和久期进行分组，统计各组的平均表现
5. **组合优化**：跨久期进行组合搭配，优化风险收益比
6. **结果展示**：生成综合分析报告和可视化图表

## 核心功能

### 1. 债券指数分类

项目支持按以下维度对债券指数进行分类：

- **按品种分类**：信用债、利率债、可转债、其他指数
- **按期限分类**：短期(0-1年)、中短期(0-3年)、中期(3-5年)等
- **按评级分类**：高评级(AAA, AA+)、中高评级(AA, AA-)等
- **按主体分类**：全市场、央企、地方国企、民企、城投

### 2. 指标计算

- **收益率**：日收益率、周收益率、月收益率、年化收益率、累计收益率
- **波动率**：历史波动率、滚动波动率
- **风险调整收益**：夏普比率、信息比率
- **最大回撤**：单日最大回撤、阶段最大回撤
- **卡尔玛比率**：年化收益率/|最大回撤|

### 3. 分组分析

- **收益率分组**：按0.5%间隔分组
- **久期分组**：按0.5年间隔分组
- **三维分组**：收益-久期-最大回撤三维分析

### 4. 投资组合优化

- 跨久期分组进行组合搭配
- 优化目标：收益高于目标、回撤低于目标
- 组合大小：3-10只指数

## 配置参数

在 `config.py` 中可以修改以下参数：

| 参数 | 默认值 | 说明 |
|------|--------|------|
| RETURN_BIN_SIZE | 0.5 | 收益率分组间隔(%) |
| DURATION_BIN_SIZE | 0.5 | 久期分组间隔(年) |
| DRAWDOWN_BIN_SIZE | 0.5 | 最大回撤分组间隔(%) |

| MIN_COMBO_SIZE | 3 | 最小组合大小 |
| MAX_COMBO_SIZE | 10 | 最大组合大小 |
| SAMPLE_SIZE | 1000 | 采样数量 |

## 数据依赖

### 输入数据

- **债券行情数据**：日度收盘价、日度收益率、成交量、成交额
- **债券基础信息**：债券代码、债券名称、发行规模、剩余期限、主体评级、债项评级
- **指数成分数据**：成分债券列表、成分权重、调样记录

### 数据库表结构

- `bond.indexcode` - 指数基本信息（代码、名称、久期）
- `bond.indexcloseprice` - 指数历史价格数据

## 注意事项

1. **数据库连接**：确保数据库凭证正确，网络连接正常
2. **数据完整性**：部分指数可能数据缺失，需要处理NaN值
3. **时间计算**：注意工作日和自然日的区别，确保日期格式一致
4. **分组边界**：边界值处理，极端值处理
