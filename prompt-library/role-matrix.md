# Prompt Library Matrix

This directory will hold role-based prompt packs for BondClaw.

## Canonical Roles

- 宏观研究员
- 利率研究员
- 信用研究员
- 转债研究员
- 多资产研究员
- 固收基金经理
- 固收交易员

## Prompt Pack Shape

Each prompt pack should include:

- role
- workflow
- variant
- required_skills
- input_schema
- prompt_template
- output_format
- sample_case
- qa_checklist

## Writing Skill

All writing packs should use the canonical `research-writing` skill.

Legacy aliases:

- `yeqing-writing`
- `zhangliang-writing`
