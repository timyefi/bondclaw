# 金融资负取数程序（简化版）使用说明

## 概述

本程序是金融资负数据取数脚本的简化版本，主要功能包括：

1. **基金净申购数据**：每天更新
2. **金融资负数据**：仅在每月20-30号更新上月数据

## 主要改进

### 相比原版的主要改进：

1. **简化逻辑**：
   - 去除了复杂的数据质量检查、验证、日志记录等功能
   - 代码从1200+行简化到约300行
   - 逻辑更加清晰易懂

2. **解决重复取数问题**：
   - 限制金融资负数据仅在每月20-30号更新
   - 获取上个月的数据并强制更新
   - 避免每天重复获取和插入数据

3. **保留核心功能**：
   - 基金净申购部分保持每天运行
   - 支持Wind和同花顺两种数据源
   - 数据库操作保持稳定

## 运行机制

### 时间控制
- **基金净申购**：每天运行，获取当日数据
- **金融资负数据**：仅在每月20-30号运行，获取上月数据

### 数据更新逻辑
1. 检查当前日期是否在更新期间（20-30号）
2. 如果是更新期间：
   - 计算上个月最后一天的日期
   - 从Wind/同花顺获取该日期的数据
   - 更新到金融资负表中
3. 更新基金净申购数据（每天都会执行）

## 使用方法

### 运行程序
```bash
python 金融资负取数_简化版.py
```

### 设置定时任务
建议设置为每天运行，程序会自动判断是否需要更新金融资负数据：

```bash
# 每天上午9点运行
0 9 * * * cd /data/项目/快速处理/图表/金融资负 && python 金融资负取数_简化版.py
```

## 代码结构

### 主要函数

1. **get_last_month_date()**：获取上个月最后一天
2. **is_update_period()**：判断是否在更新期间
3. **update_financial_data()**：更新金融资负数据
4. **update_database()**：更新数据库
5. **update_fund_data()**：更新基金净申购数据
6. **main()**：主函数

## 注意事项

1. **数据源**：
   - Wind数据代码：使用Wind EDB接口
   - 同花顺数据代码：使用同花顺EDB接口

2. **时间窗口**：
   - 金融资负数据仅在每月20-30号更新
   - 基金净申购数据每天更新

3. **错误处理**：
   - 程序包含基本的错误处理机制
   - 遇到错误会输出错误信息但不会中断整个程序

4. **性能考虑**：
   - 在API调用之间添加了延迟，避免请求过于频繁
   - 简化了数据库操作，提高执行效率

## 维护说明

### 添加新的数据代码
1. Wind数据：添加到`wind_codes`列表
2. 同花顺数据：添加到`ths_codes`列表

### 修改更新时间
修改`is_update_period()`函数中的日期判断逻辑

### 数据库表结构
- 主表：`金融资负` - 存储金融资负数据
- 基金表：`基金净申购` - 存储基金净申购数据

## 常见问题

### 1. 数据没有更新
- 检查是否在更新期间（20-30号）
- 检查API接口是否正常
- 检查数据库连接是否正常

### 2. 重复数据问题
- 本版本已解决重复数据问题
- 通过时间控制确保每月只更新一次

### 3. 性能问题
- 程序已优化，执行时间应该在合理范围内
- 如有性能问题，可检查网络连接或API响应速度
