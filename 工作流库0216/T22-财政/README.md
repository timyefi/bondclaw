# 财政

## 概述

财政数据整合导入项目，用于读取、清洗和导入中国财政数据到数据库。支持多年度、多维度的财政数据管理。

**主要功能**：
- 财政数据读取（Excel格式）
- 数据清洗和转换
- 地域信息解析（省级/地市级）
- 数据库表创建和数据导入
- 多年份批量处理

**应用场景**：
- 财政政策研究
- 地方债务分析
- 宏观经济研究
- 城投债风险评估

## 数据维度

### 债务相关指标
| 指标 | 说明 |
|------|------|
| 负债率(%) | 债务余额/GDP |
| 负债率(宽口径)(%) | 含隐性债务 |
| 债务率(%) | 债务余额/综合财力 |
| 城投债余额(亿) | 城投债券存量 |
| 地方政府债余额(亿) | 地方债存量 |
| 城投有息负债(亿) | 城投有息债务总额 |

### 财政收支指标
| 指标 | 说明 |
|------|------|
| 一般公共预算收入(亿) | 财政收入 |
| 一般公共预算支出(亿) | 财政支出 |
| 财政自给率(%) | 收入/支出 |
| 税收收入(亿) | 税收收入 |
| 政府性基金收入(亿) | 基金收入 |
| 国有土地权出让收入(亿) | 卖地收入 |

### 经济指标
| 指标 | 说明 |
|------|------|
| GDP(亿) | 地区生产总值 |
| GDP增速(%) | 经济增长率 |
| 人均GDP(元) | 人均产出 |
| 房地产投资(亿) | 房地产开发投资 |
| 商品房销售面积(万平方米) | 房地产销售 |
| 常住人口(万) | 人口规模 |

## 数据源

- **数据文件**: `data/17-24chinafiscaldata.xlsx`
- **数据范围**: 2017-2024年全国省级和地市级财政数据
- **数据格式**: Excel文件，每年一个工作表

## 输出结果

- **数据库表**: `china_fiscal_data_2017_2024`
- **记录数**: 取决于Excel数据量
- **覆盖范围**: 2017-2024年全国省级和地市级财政数据

## 注意事项

1. 运行前确保数据库连接配置正确
2. Excel文件格式需要符合预期
3. 支持增量导入（通过年份工作表名称判断）
4. 地域名称需要与标准省名匹配
5. 大文件（>10MB）未复制，需手动处理
6. 敏感信息请使用环境变量
